자주 묻는 질문

트래픽이 늘면 비용이 비례해서 늘어나나요?

대부분의 클라우드 비용은 비선형적으로 증가합니다. 특히 LLM API 비용은 사용량에 정비례하지만, 서버와 DB는 스케일링 방식에 따라 다릅니다.

모니터링 비용도 포함해야 하나요?

네. Datadog, CloudWatch 등 모니터링 도구 비용은 인프라 규모에 따라 월 수백 달러에 달할 수 있어 빠뜨리면 예산 차이가 큽니다.

초기 단계에서 가장 큰 비용 항목은?

대부분 LLM API 비용이 가장 큽니다. 서비스 성장 후에는 GPU 서버와 DB 비용이 급격히 늘어나는 경우가 많습니다.
