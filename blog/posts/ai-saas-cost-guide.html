팀 크기에 따라 뭘 골라야 할까

1인 개발자·프리랜서

  • 추천 조합: Claude Pro ($20) + GitHub Copilot ($10)
  • 월 비용: 약 43,000원 (환율 1,450원 기준)
  • ChatGPT와 Claude를 둘 다 쓸 필요는 없습니다. 개인적으로는 코딩 비중이 높다면 Cursor 하나로 합치는 게 낫다고 봅니다

5인 이하 소규모 팀

  • 추천 조합: Claude Team + GitHub Copilot Business
  • 월 비용 (5인 기준): 약 $220/월 → 약 319,000원
  • Notion AI는 이미 Notion을 쓰고 있는 팀만 추가할 가치가 있습니다. 아니라면 굳이 안 써도 됩니다

10-20인 스타트업

  • 핵심 도구 2-3개만 팀 전체 구독하고, 나머지는 실제로 쓰는 사람만 결제하세요
  • 전원이 ChatGPT+Claude를 이중 구독하는 건 거의 확실하게 낭비입니다. 용도를 나눠야 합니다
  • Microsoft 365를 이미 구독 중이라면 Copilot 추가가 가성비 면에서 가장 깔끔합니다

숨은 비용을 놓치면 안 된다

1. 중복 구독 제거

생각보다 ChatGPT와 Claude를 동시에 구독하는 사람이 많습니다. 한번 점검해 보세요. 진짜 둘 다 쓰고 있나요? 주력 도구 하나로 통합하면 월 $20가 바로 남습니다.

2. 연간 결제 전환

대부분의 AI SaaS는 연간 결제 시 15-20% 할인을 제공합니다. 6개월 넘게 꾸준히 쓴 도구라면 연간 플랜으로 바꾸는 게 이득입니다. 다만 AI 도구 시장이 워낙 빠르게 바뀌다 보니, 개인적으로는 확신이 없으면 월간 유지를 권합니다.

3. API vs 구독 판단

매일 쓰지 않는다면 API 종량제가 더 쌀 수 있습니다. Claude API를 월 $5-10 수준으로 쓴다면 Pro 구독($20)의 절반도 안 드는 셈이니까요.

구독 vs API 기준: 하루 20회 미만 사용이면 API 종량제, 하루 20회 이상 집중적으로 사용하면 구독이 경제적입니다.

자주 나오는 질문

Q. ChatGPT와 Claude 중 하나만 써야 한다면 어떤 걸 선택해야 하나요?

용도에 따라 다릅니다. 코드 작성·긴 문서 분석·정밀한 글쓰기는 Claude가 강점이 있고, 이미지 생성(DALL-E)·플러그인 생태계·GPTs 활용은 ChatGPT가 유리합니다. 개발자라면 Claude, 다양한 생성 AI 기능이 필요하다면 ChatGPT를 추천합니다.

Q. GitHub Copilot과 Cursor 중 어느 게 더 나은가요?

현재 사용하는 IDE가 기준입니다. VS Code·JetBrains 등 기존 IDE를 계속 쓰고 싶다면 Copilot, AI 기능 최우선의 새로운 IDE 경험을 원한다면 Cursor가 낫습니다. Cursor는 Claude를 내장해 더 정밀한 코드 편집이 가능하지만, 기존 환경에서 벗어나야 하는 비용이 있습니다.

Q. 스타트업에서 AI 도구 구독을 비용 처리할 수 있나요?

네. 업무와 직접 관련된 SaaS 구독료는 소프트웨어 비용으로 손금 처리가 가능합니다. 법인카드로 결제하고 세금계산서(또는 영수증)를 보관하면 됩니다. 개인 명의 계정이라면 환급 처리가 복잡해질 수 있으므로 가능하면 법인 계정으로 구독하는 것이 좋습니다.

Q. AI 도구가 너무 빠르게 바뀌는데 어떻게 선택해야 하나요?

연간 구독 대신 월간 구독으로 유연성을 유지하는 것이 좋습니다. 또한 핵심 도구 1-2개만 유료 구독하고, 나머지는 무료 플랜이나 API로 실험적으로 써보다가 업무에 실제로 필요하다고 판단될 때 구독을 추가하는 방식이 가장 현명합니다.

핵심만 뽑으면

AI 도구는 분명 생산성을 올려줍니다. 하지만 구독료가 쌓이면 이야기가 달라집니다. 분기에 한 번만 구독 목록을 열어서, 월 10회도 안 쓰는 도구는 바로 끊으세요. 중복 제거와 연간 결제 전환, 이 두 가지만 해도 팀 단위로 연간 수백만 원이 남습니다.
