# 블로그 글 품질 기준

글 완성 전 아래 체크리스트를 모두 통과해야 함.

## 분량

- 순수 텍스트 2,500자 이상 (HTML 태그 제외한 실제 읽히는 텍스트)
- 실제 프로젝트 파일 기준: 6,000바이트(HTML 포함) 이상이면 대부분 충족

## 인물 기반 계산 예시

- 3-4개의 구체적 인물 예시 포함
- 형식: 이름/나이/직업/상황을 명시한 시나리오 → 수치 계산 결과
- 예: "30대 직장인 김OO씨, 연봉 4,500만 원 기준으로..."
- 추상적 설명만 있는 글 금지

## FAQ

- HTML `<div class="faq-item">` 4-5개
- JSON-LD FAQPage 스키마와 Q&A 내용 일치 필수
- 실제 독자가 검색할 법한 질문 (예: "XX는 어떻게 계산하나요?")

## 교차 링크

- 관련 계산기 링크: 최소 1개 (CTA 박스 포함)
- 관련 글 또는 추가 계산기 링크: 최소 1개 추가
- 합계 최소 2-3개

## 금지 사항

- "안녕하세요! 오늘은..." 식의 AI 전형적 인사 도입부 금지
- 하이픈(-)으로 시작하는 리스트 항목 금지 (`<ul><li>` 사용)
- em dash(—) 금지, 하이픈(-) 사용
- 막연한 "다양한", "여러 가지" 표현 남발 금지 - 구체적 수치/사례로 대체
