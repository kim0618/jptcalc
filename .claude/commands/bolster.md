## 작업 순서

## 보강 방법 (추가할 콘텐츠 유형)

기존 구조를 건드리지 않고, **새 h2 섹션을 본문 중간~후반에 삽입**한다.
아래 유형 중 부족한 글자 수에 맞게 2~3개를 조합한다.

### 유형 1: 반전 비교 섹션 (약 1,500~2,500자)
독자 상식을 뒤집는 비교 데이터. 가장 효과적.
- h2 제목: "~인 줄 알았는데", "여기서 함정이 있다"
- 수치 근거 필수 (계산 과정 포함)
- 예) "BMR만 보면 위험한 이유 - 활동량 무시하면 매일 500kcal 부족"

### 유형 2: 추가 시나리오 (약 1,000~2,000자)
기존 시나리오와 조건이 다른 새 사례 추가.
- 인물·나이·상황 구체적으로
- 기존 시나리오와 대비되는 조건 (예: 같은 연봉인데 야근 유무 차이)
- "그래서 뭐가 달라지는데" 의사결정 연결 문장 필수

### 유형 3: 비교표 추가 (약 800~1,500자)
기존에 없는 관점의 비교표 1개 추가.
- 반드시 실제 수치 포함
- 설명 문단 2~3문장 함께 작성

### 유형 4: 실전 팁/주의사항 섹션 (약 1,000~1,500자)
실무에서 자주 실수하는 점이나 독자가 모르는 꿀팁.
- h2 제목: "~할 때 이것만 조심하면 된다", "의외로 모르는 사람이 많다"
- 구체적 행동 지침 (추상적 조언 금지)

### 유형 5: 목차형 요약 리스트 추가 (약 500~800자)
기존 글에 목차형 요약이 없으면 도입부 직후에 삽입.
```html
<ul style="background:#F9FAFB;padding:16px 20px 16px 36px;border-radius:10px;margin:16px 0;font-size:14px;color:#374151;line-height:1.8">
  <li>항목 1</li>
  <li>항목 2</li>
  <li>항목 3</li>
  <li>항목 4</li>
</ul>
```

### 유형 6: 신뢰 시그널 보강 (약 300~500자)
기존 본문에 법조문명, 공식 통계, 고시명 인용이 없으면 추가.
- 기존 문단에 자연스럽게 삽입 (별도 섹션 불필요)
- "근로기준법 제56조에 따르면~", "2025년 통계청 조사 기준~"

## 절대 변경하지 않는 것
- 기존 데이터/표/수치
- HTML 구조 (CSS 클래스, 태그, 레이아웃)
- 기존 내부 링크, CTA 버튼
- JSON-LD, script 블록
- SVG 인포그래픽
- h1 제목, meta description, title

## 변경해도 되는 것
- 새 h2/h3 섹션 삽입
- 기존 문단에 1~2문장 추가 (신뢰 시그널 등)
- highlight-box 추가
- 새 비교표 추가

## 글쓰기 금지사항
- em dash(—) 사용 금지, 하이픈(-) 사용
- "계산기으로" 사용 금지, "계산기로"가 올바른 표현
- "공인인증서" 사용 금지, "공동인증서"로 작성
- "~정리합니다", "~알아보겠습니다" 남발 금지 (전체 글에서 1회 이하)
- 분량 채우기용 중복·반복 문장 금지
- 수치 없는 추상적 설명 금지
