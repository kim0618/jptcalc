**b) 자주 찾는 계산기 (3개 shortcut)**
- 해당 카테고리 계산기 중 검색량 상위 3개를 대표 카드로

**c) 상황별 추천 계산기 (4-6개 시나리오)**

**d) 그룹별 계산기 목록 (2-3개 그룹)**

- 계산기 카드당 **2-3문장 설명** (기준·수식·활용)
- 현재는 1줄인 설명을 구체화

**e) 카테고리 배경 지식 섹션 (1,500-2,500자)**

- 법령·공식·기준 배경 (근거 명시: "근로기준법 제56조", "2026년 국세청 고시" 등)

### 3) JSON-LD 갱신 (head 안, 필수)

기존 WebApplication + FAQPage에 **CollectionPage 추가**, FAQPage의 질문을 **본문 FAQ와 1:1 매칭**으로 갱신.

```json
{
  "@context": "https://schema.org",
  "@type": "CollectionPage",
  "name": "{카테고리} 계산기",
  "url": "https://www.jptcalc.kr/calc/{slug}/",
  "description": "...",
  "inLanguage": "ko-KR",
  "hasPart": [
    { "@type": "WebApplication", "name": "계산기명", "url": "..." }
  ],
  "breadcrumb": {
    "@type": "BreadcrumbList",
    "itemListElement": [
      { "@type": "ListItem", "position": 1, "name": "홈", "item": "https://www.jptcalc.kr/" },
      { "@type": "ListItem", "position": 2, "name": "{카테고리} 계산기", "item": "https://www.jptcalc.kr/calc/{slug}/" }
    ]
  }
}
```
