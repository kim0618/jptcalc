# 제이퍼 계산기 정책 수치 마스터 데이터

`/jptcalc-refresh` 스킬의 마스터 데이터. 변동 가능한 세율·요율·기준값을 추적한다.

**최초 생성**: 2026-04-29
**마지막 일제 점검**: -

## 항목 구조

각 항목은 다음 필드를 가진다.
- **현재 값**: 최신 확인된 값
- **적용 시기**: 어느 연도/시점 기준인지
- **공식 출처**: 검증 시 WebFetch할 URL
- **마지막 확인**: YYYY-MM-DD
- **변동 주기**: 연 1회 / 분기 / 한은 결정 시 / 법 개정 시
- **검색 패턴**: 영향 페이지 식별용 grep 패턴 (여러 표기 형태)
- **영향 페이지**: 첫 가동 시 자동 매핑
